# 💍 София & Никита — Свадебное приглашение

Роскошное, кинематографическое электронное приглашение на свадьбу.
**1 августа 2026 года.**

## ✅ Реализованные секции

- **Loading Screen** — анимированные золотые кольца с монограммой С&Н
- **Hero** — полноэкранный экран с canvas particle-системой (лепестки + orbs + sparkles), золотые имена, SVG-рамка, параллакс
- **О нас** — романтическая история с плавающими blob-декорациями, пульсирующее сердце
- **Countdown** — живой обратный отсчёт с карточками и particle canvas
- **Save the Date** — финальный экран со звёздным canvas и хэштегом
- **Footer** — монограмма, дата, финальное послание

## ✨ Визуальные эффекты

- 3 canvas particle-системы (hero, countdown, savedate)
- Cursor glow + sparkle trail (desktop)
- Scroll reveal (Intersection Observer)
- Gold shimmer градиент на заголовках
- Floating blob декорации
- Heartbeat пульсация
- Parallax hero
- SVG рамка с ботаническими листьями
- Кнопка музыки с эквалайзером
